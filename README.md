# smartSchool
Ανάπτυξη ενός ασύρματου δικτύου αισθητήρων στον σχολικό χώρο, με σκοπό τη μέτρηση και καταγραφή της θερμοκρασίας, της υγρασίας και της φωτεινότητας σε επιλεγμένους χώρους του σχολείου.

Για την υλοποίηση θα χρησιμοποιηθούν αυτόνομες μονάδες αισθητήρων, βασισμένες σε μικροελεγκτές, οι οποίες θα επικοινωνούν ασύρματα με τον σταθμό βάσης μέσω δικτύου LoRa. Η συγκεκριμένη τεχνολογία επιλέγεται λόγω της μεγάλης εμβέλειας και της χαμηλής ενεργειακής κατανάλωσης, εξασφαλίζοντας αξιόπιστη μετάδοση δεδομένων σε ολόκληρο το σχολικό κτίριο.

Τα δεδομένα που θα συλλέγονται θα προβάλλονται μέσω απλής ιστοσελίδας και εφαρμογής για κινητές συσκευές, επιτρέποντας την εύκολη παρακολούθηση των συνθηκών στους χώρους του σχολείου. Η αξιοποίηση των μετρήσεων συμβάλλει στον καλύτερο έλεγχο της ενεργειακής κατανάλωσης και στη γενικότερη περιβαλλοντική ευαισθητοποίηση της σχολικής κοινότητας.
