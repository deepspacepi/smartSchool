Τα αρχεία του συγκεκριμένου φακέλου περιλαμβάνουν τον κώδικα για τον αισθητήρα που βρίσκεται σε κάθε αίθουσα. Για την λειτουργία κάθε νέου αισθητήρα απαιτείται η τροποποίηση των αντίστοιχων μεταβλητών στο αρχείο config.h σύμφωνα με τα στοιχεία που δίνονται από το TTN.

Για την δημιουργία του αισθητήρα απαιτούνται τα παρακατώ υλικά.

* Waveshare RP2040-LoRa-HF with SX1262 RF & USB Type-C Adapter
* Μπαταρία Λιθίου 18650 3350mAh
* Μπαταριοθήκη 18650
* Αισθητήρας Υγρασίας & Θερμοκρασίας AHT10
* Light Sensor - BH1750
* Lithium Battery Charger and Protection Module 1A USB-C - TP4056
* Διάφορα Αναλώσιμα

Η συνδεσμολογία που απαιτείται για τον αισθητήρα φαίνεται στην επόμενη εικόνα.
