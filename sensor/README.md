Τα αρχεία του συγκεκριμένου φακέλου περιλαμβάνουν τον κώδικα για τον αισθητήρα που βρίσκεται σε κάθε αίθουσα. Για την λειτουργία κάθε νέου αισθητήρα απαιτείται η τροποποίηση των αντίστοιχων μεταβλητών στο αρχείο config.h σύμφωνα με τα στοιχεία που δίνονται από το The Things Network (TTN).

Για την δημιουργία του αισθητήρα απαιτούνται τα παρακατώ υλικά.

* Waveshare RP2040-LoRa-HF with SX1262 RF & USB Type-C Adapter
* Μπαταρία Λιθίου 18650 3350mAh
* Μπαταριοθήκη 18650
* Αισθητήρας Υγρασίας & Θερμοκρασίας AHT10
* Light Sensor - BH1750
* Lithium Battery Charger and Protection Module 1A USB-C - TP4056
* Διάφορα Αναλώσιμα

Η συνδεσμολογία που απαιτείται για τον αισθητήρα φαίνεται στην επόμενη εικόνα.

Για την σύνδεση του αισθητήρα με το TTN απαιτείται πρώτα η δημιουργία μίας εφαρμογής (Application). Η δημιουργία της είναι απλή και θα πρέπει μόνο να συμπληρώσουμε Application ID Application name όπως φαίνεται στην επόμενη εικόνα.

<img align="center" width="1920" height="1080" alt="TTN-Application" src="https://github.com/user-attachments/assets/1bac2aa5-251f-4e59-9221-081d465b6623" />

Στην συνέχεια θα πρέπει να δημιουργήσουμε μέσα στο Application όλες τις τερματικές συσκευές που θέλουμε να συνδέσουμε για την αποστολή δεδομένων. Στην επόμενη εικόνα φαίνονται οι βασικές ρυθμίσεις που επιλέχθηκαν για τον αισθητήρα μας καθώς και οι πληροφορίες που δημιουργήθηκαν για την σύνδεση του με το TTN.



Συγκεκριμένα, θα πρέπει να συμπληρώσουμε το joinEUI που δίνεται από τον κατασκευαστή, ή σε περίπτωση που δεν δίνεται κάτι μπορούμε να συμπληρώσουμε με μηδενικά. Επιπλέον, ως DevEUI θα πρέπει να συμπληρώσουμε την τιμή που δίνεται από τον κατασκευαστή και μπορεί να βρεθεί με την εκτέλεση του αντίστοιχου προγράμματος. Όσον αφορά τα AppKey, NwkKey μπορούμε να συμπληρώσουμε δικές μας τιμές ή να πατήσουμε Generate για να δημιουργηθούν αυτόματα. Τέλος, ως End device ID συμπληρώνουμε το επιθυμητό ID για την τερματική συσκευή μας.

Να σημειώσουμε ότι όλες αυτές τις πληροφορίες θα πρέπει να τις συμληρώσουμε και στο αρχείο config.h. Για ευνόητους λόγους ασφαλείας, στα παραδείγματα μας δεν έχουμε πραγματικές τιμές.
